☕ Coffee Bean Scanner
A full-stack deep learning application that classifies coffee beans from images using a PyTorch CNN model, served through a Python backend and an interactive frontend.

🧠 Project Overview
Coffee bean quality is traditionally assessed by hand — a slow, inconsistent, and labor-intensive process. This project automates that using computer vision. Upload an image of a coffee bean and the model predicts its roast level or quality class in real time.
The system is end-to-end: raw dataset → trained model → REST API backend → browser frontend.
